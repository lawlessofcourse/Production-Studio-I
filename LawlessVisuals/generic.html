L.I.P.S.

Lawless Interactive Performance System

Project Proposal

Abstract

My project idea is to build upon my performance system I have been building. The system uses a variety of programs with the goal of mixing audio reactive and audio driven visuals in a live performance setting. So far, I use Resolume to mix the visuals, digital media, and processing code live. This gives me the ability to add audio reactive effects to regular motion graphics. This requires creating content in any fashion. This semester I would like to work on creating more content and integrating Isadora into the mix by exploring its possibilities. I might work with Trason and develop some content that we can mix live with audio reactive effects.

Introduction to the project/Background on the System

This system uses a variety of programs with the goal of mixing audio reactive and audio driven visuals in a live performance setting. So far, I use Resolume to mix the visuals, digital media, and processing code live. Resolume “is an instrument for VJs, AV performers and video artists. It puts all your media and effects right at your fingertips so you can quickly improvise your live visuals (Resolume).” This gives me the ability to add audio reactive effects to regular motion graphics. During the semester I had developed the same idea of a project in two different fashions. The first iteration I used Processing, which is a creative Integrated Development Environment for Java programmers. In this application I designed a program to analyze sound being played live or through an audio file. Then I would send out shapes and control them with the data from the sound file, or the sound would generate shapes itself. I was also able to connect certain parameters to my Midi controller. This was the basis of me becoming a VJ. For those who do not know, a VJ’s job is to present visuals that represent or portray an artist’s performance.

Detailed description of the proposed project

This semester I would like to focus on developing content for my system. While doing that I hope to find a creative path that I can use to create detailed visual content easily. I would like to develop content using Maya. For the content, it will consist of 3D rendered pieces that I can bring into resolume and mix seamlessly. In Maya, I plan to develop animations that can be mapped to music through resolume and other pathways. I would also like to create a rendered piece to a specific song, from which I can draw content to use for other projects and performances. Essentially, I want to create VJ Loops and one animation/VJ mix to a certain song.

In terms on animation I would love to learn how to capture motion data from a live performer. It would be really fun if I could capture motion data from the real world and animate characters with the motion data and an auto rigger.

First Iteration for Interactive Performance Systems

SPRING 2019 First Iteration

Everything is automated, I did nothing but hit play.

This is the first iteration of my VJ project for Interactive Performance Media during the Spring of 2019 at the University of Montana. This project uses processing to automate shapes reaction(s) to the music. The original project had Midi control that I used to control objects live while being played. I also had a few more elements in there that I had to take out. This version is fully automated and driven by the audio. The original had almost identical audio reactions but some additional elements, background color, and videos. Once I get the chance I would like to record a preformed version on a better computer so the graphics can keep up.

Related Work and Background

Where it all started

It all started with me playing around in After Effects and using the Trapcode Suite plug in from Red Giant. In the suite they have a lot of audio reaction tools to use with their systems. This is where I started and fell in love with making audio reactive and audio driven art. It pushed me to experiment with different designs and concepts and eventually pushing me into a new realm developing audio reactive visuals.


The reason this is important to my system is due to my fascination with creating audio reactive art. In an ideal system I would be able to create pieces like this but live and in real time. All of this work shown in this section has been designed, created, and rendered out to a specific song. And that's when it dawned on me. What if I could create visuals like this but have them react in real time to music that is being played live? The spawn of this system stemmed from the creation of these rendered pieces.


Made this digital piece in After Effects using the Trapcode Suite and by following a Red Giant Tutorial. Song is Outlaws by The Widdler.

I made this by following another tutorial by Red Giant. This uses their Sound Keys plugin which lets you analyze a sound file using FFT. You can then measure the data from the FFT and use that to control aspects of your design in After Effects. Another example of me diving into audio reaction.


This is a more recent personal project I did in After Effects using the Trapcode Suite. In this one I wanted to experiement with rigged 3D models and audio reaction. I made the character in adobe fuse, brought it mixamo where I auto rigged the girl to do a catwalk. From there I made an obj sequence and designed the rest in after effects.

Project Significance

After graduation I would love to become a VJ for a musician or work on a creative team that develops interactive systems, installations, and stage setups. I still believe I need to develop some more coding skills in other languages. I also wish to work on my animation and 3d animation.

This system will hopefully show that I have the resources and know how to work in such an environment that I would encounter either VJing or working for a creative team.
